
1.Name of your Project ?
ANS:
Premium Digital Tools Platform.

2.A little Description?
ANS:
এটি একটি আধুনিক ওয়েব অ্যাপ্লিকেশন যার নাম DigiTools। এখানে ব্যবহারকারীরা বিভিন্ন ধরনের প্রিমিয়াম ডিজিটাল রিসোর্স (যেমন: রেজ্যুমি বিল্ডার, পোর্টফোলিও কিট এবং জব ট্র্যাকার) ব্রাউজ করতে পারবেন। এতে একটি চমৎকার ইউজার ইন্টারফেস এবং ইন্টারঅ্যাক্টিভ কার্ট (Cart) ম্যানেজমেন্ট সিস্টেম রয়েছে।

3.Technology that you use ?
ANS:
1.React.js.
2.Tailwind CSS, DaisyUI.
3.JavaScript (ES6+).
4.React-Toastify (NPM Package),React Icone.
5.JSON (for product data).
6.Vite 8(React 19).

3 Features about your project?
ANS:
Dynamic Cart System: Users can add multiple products to the cart, and the count updates in real-time.

Interactive Notifications: Uses Toast messages to give feedback (Success/Warning) when a user tries to checkout.

Responsive UI: The entire platform is fully responsive, meaning it looks great on both mobile and desktop screens.
